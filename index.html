Raspberry Pi

Arduino/Genuino

 Raspberry Pi 3

  • ARM Cortex-A53 - 64bit ARMv8A
  • 1GB LPDDR2 (900 MHz)
  • 10/100 Ethernet, 2.4GHz 802.11n, Bluetooth 4.1
  • HDMI, 3.5mm analogue audio-video jack, 4× USB 2.0, Camera Serial Interface (CSI), Display Serial Interface (DSI)
